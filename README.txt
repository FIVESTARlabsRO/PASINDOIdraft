PAS ÎN DOI — Website Demo
=========================
Creat de FivestarLabs

CUM SCHIMBI IMAGINILE:
----------------------
1. Imaginea principala (hero/banner):
   - Inlocuieste fisierul: images/hero.jpg
   - Foloseste o fotografie frumoasa a salii sau restaurantului
   - Dimensiune recomandata: 1920x1080px minim

2. Imaginea sectiunii Evenimente:
   - Inlocuieste fisierul: images/events.jpg
   - Foloseste o fotografie dintr-un eveniment organizat
   - Dimensiune recomandata: 1920x600px minim

IMPORTANT: Numele fisierelor trebuie sa ramana exact la fel.
Doar inlocuieste fisierul cu unul nou care are acelasi nume.

CUM PUBLICI PE GITHUB PAGES:
-----------------------------
1. Creeaza un repository nou pe github.com
2. Urca toate fisierele (index.html + folderul images/)
3. Mergi la Settings > Pages > Source: main branch
4. Site-ul e live la: username.github.io/repository-name
